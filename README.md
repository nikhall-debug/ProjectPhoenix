# 🔥 Project Phoenix

Project Phoenix is a personal health and performance intelligence system.

Its purpose is not simply to collect health data, but to understand it.

Phoenix combines data from multiple health and fitness platforms to learn how **Nik Hall's** body responds to training, nutrition, recovery, illness and lifestyle.

## Core Principles

- Manual input must take less than 30 seconds.
- Everything possible should be imported automatically.
- The AI should explain *why*, not just display numbers.
- Long-term trends are more important than daily fluctuations.
- Phoenix learns Nik's physiology rather than comparing him with population averages.
